## RemoveBlank2

- Continuous blank lines are deleted when saving the document.
- for VisualStudio 2019 Extension

<br>

## Features 
```
replace

from: (^(?([^\r\n])\s)*\r?\n){2,}
to :  Environment.NewLine ( or \r\n or \n)

on save.
```
